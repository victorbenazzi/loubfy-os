# Gemini Guide

Read `AGENTS.md` first. It is the source of truth for this project.

Loubfy OS is a reusable social media and blogpost content operating system guided by AI agents. Loubfy OS is the project/product name, not the onboarded business name.

Use `skills/{skill-name}/SKILL.md` as the process layer. For complex work, read `workflows/agent-orchestration.md` and use the agent cards in `agents/roster/` as role guidance. If Gemini, Antigravity, or the current terminal interface does not support subagents, execute the same roles sequentially.

Save content under `content/`, store image assets under `images/`, and keep durable facts in `memory/`.

If any instruction here conflicts with `AGENTS.md`, follow `AGENTS.md`.
