# General Guidelines

Use this file for reusable content production rules.

## Content Principles

- Every piece needs a clear thesis.
- Every post should know its platform before it is written.
- Strategy, copy, and visual direction should be saved separately.
- References and sources should be recorded when used.
- Long-form content should be repurposed into social assets.

## Production Flow

1. Define context.
2. Gather or reference research.
3. Choose content angle.
4. Write copy.
5. Build visual brief.
6. Save final output.
7. Record performance.

## Folder Rule

Create one folder per content piece:

```text
content/social-media/{platform}/{slug}/
```
