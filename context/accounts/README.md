# Accounts

Use esta pasta para contexto específico de cada conta/perfil social administrado pelo projeto.

Formato recomendado:

```text
{platform}-{accountId}.md
```

Exemplos:

```text
instagram-empresa-institucional.md
instagram-fundador.md
linkedin-ceo.md
```

`accountId` deve ser o mesmo usado em:

```text
content/social-media/{platform}/{accountId}/{slug}/post.json
```

Use a mesma raiz de projeto quando as contas compartilham empresa, branding base, oferta e aprendizados. Use outro projeto quando a operação for de outra marca/cliente.
