# Branding

Use this file for brand assets and identity rules.

## Logos

Store logo files in:

```text
brand/logos/
```

## Fonts

Store font files or font notes in:

```text
brand/fonts/
```

## Brand Usage

- Logo placement: for Instagram editorial templates, use top metadata text instead of forcing a logo into every slide.
- Minimum clear space: keep the 60 px top/bottom and 40 px side safe area clear for text and metadata.
- Background rules: default system is black/white; client onboarding colors may replace template tokens.
- Image treatment: use real or licensed photos as backgrounds, with readable gradient overlays for text-heavy slides.
- Do not use: the supplied orange reference as the default Loubfy OS color; it is a layout reference only.
