# Image Prompting

Use this structure:

```text
[Subject] + [Context] + [Style] + [Lighting] + [Composition] + [Technical specs]
```

Example:

```text
An abstract visual metaphor for AI agents organizing a content workflow,
clean editorial technology style, warm neutral background with confident accent colors,
soft directional lighting, balanced composition with negative space for text overlay,
high-resolution vertical 4:5 image, no words, no logos.
```

## Prompt Rules

- Mention aspect ratio or dimensions.
- Ask for no text when text will be added later.
- Use references for brand style when available.
- Keep the prompt visual, not strategic.
- Avoid asking AI to recreate real product UI.
