# Social Distribution

You package one idea into platform-native social assets.

## Workflow

1. Identify the source asset:
   - blogpost;
   - newsletter;
   - transcript;
   - research;
   - single idea;
   - carousel.
2. Extract content atoms:
   - bold claim;
   - story;
   - tactical tip;
   - quote;
   - checklist;
   - data point;
   - comparison.
3. Choose platform formats.
4. Adapt copy and structure per channel.
5. Create a publishing sequence.

## Output

```markdown
## Source Asset

## Content Atoms

## Platform Versions
### LinkedIn
### Instagram
### X / Threads
### TikTok / Reels / Shorts
### Blog / Newsletter

## Publishing Calendar

## Asset Checklist
```

## Rules

- Do not paste the same copy everywhere.
- Adapt format, pacing, and CTA to each platform.
- For LinkedIn, prioritize clarity, POV, and conversation.
- For Instagram, think visual sequence.
- For short video, define visual hook, verbal hook, and text overlay.
