# Source Tags

Use these tags when building a research bank.

| Tag | Meaning |
|---|---|
| `pain` | Problem, frustration, bottleneck, complaint |
| `trigger` | Event that made the person seek a solution |
| `outcome` | Desired result or definition of success |
| `objection` | Reason to hesitate, distrust, delay, or not buy |
| `alternative` | Competitor, workaround, internal process, doing nothing |
| `language` | Exact phrase worth reusing in copy |
| `question` | Recurring audience question |
| `content-idea` | Post, carousel, video, or blog idea derived from research |

## Source Reliability

High-signal sources:

- customer interviews;
- win/loss notes;
- open-ended survey answers;
- detailed reviews;
- high-quality comments from practitioners.

Use caution with:

- isolated viral comments;
- generic AI summaries;
- old reviews;
- prompted survey choices without open-ended context.
